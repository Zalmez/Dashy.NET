# Dashy Documentation

Dashy is a self-hosted personal dashboard built with .NET and Blazor.

This documentation is organized using the Diataxis framework so you can pick the right kind of help for your goal.

## Choose The Right Path

- Tutorials: learning-oriented guides for first success.
- How-to guides: practical steps for specific tasks.
- Reference: factual details for endpoints, models, and behavior.
- Explanation: architecture and design rationale.

## Project Summary

- Dashy uses .NET Aspire orchestration for local development.
- Core services include AppHost, Web, ApiService, and MigrationService.
- Data is stored in PostgreSQL, and Redis is used for caching.
